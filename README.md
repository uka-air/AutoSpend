# AutoSpend (React Native)

A React Native (Expo) starter for iOS and Android that can:

- Import receipt images from gallery
- Take receipt photos with camera
- Run OCR text extraction
- Let users manually review OCR output (amount, merchant, category, save)
- Maintain an expense timeline with category filter and daily/monthly totals
- Track basic insights (this month spend, top category, average daily spend, recent merchants)
- Flag potential duplicates between receipt and email sourced transactions

## Categories

- food
- transport
- shopping
- bills
- groceries
- health
- entertainment
- subscriptions
- other

## Run

```bash
npm install
npm run start
```

Then launch on iOS/Android simulator or device from the Expo UI.
